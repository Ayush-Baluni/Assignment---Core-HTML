Full Stack Web Development Tech Stack


HTML:
HTML stands for 'Hyper Text Markup Language'. It is used to make the structure of the webpage. It allows embedding multimedia like images, videos, and also elements like headings, paragraphs, forms.
CSS:
CSS stands for 'Cascading Style Sheets'. CSS is used to manage style and layout of web pages. It integrated with HTML and enables responsive web design techniques for different screen shapes and sizes. We can also edit colors, fonts, spacing and much more using this and it supports responsive design techniques for different screen sizes and devices.
Tailwind CSS:
Tailwind CSS is an open-source CSS framework. The main feature of this library is that, unlike other CSS frameworks like Bootstrap, it does not provide a series of predefined classes for elements such as buttons or tables. Instead, it creates a list of 'utility' CSS classes that can be used to style each element by mixing and matching.
JavaScript:
JavaScript is a lightweight interpreted programming language with first-class functions. While it is most well-known as the scripting language for Web pages, many non-browser environments also use it, such as Node.js, Apache CouchDB and Adobe Acrobat. We can manage client-side and server-side development using JavaScript.
ReactJS:
React is a free and open-source front-end JavaScript library for building user interfaces. It is maintained by Meta and a community of individual developers and companies. React can be used to develop single-page, mobile, or server-rendered applications with frameworks like Next.js.
ExpressJS:
Express is a back-end web application framework for building APIs with Node.js. It is designed for building web applications, mobile applications and APIs. It is an open source framework developed and maintained by the Node.js foundation.
MongoDB:
MongoDB is a source-available, cross-platform, document-oriented database program. It stores data in a type of JSON format called BSON. MongoDB is developed by MongoDB Inc. and current versions are licensed under the Server Side Public License (SSPL).
Node.js:
Node.js is a free, open-source, cross-platform JavaScript runtime environment that lets developers create servers, web apps, command line tools and scripts. It runs on the V8 JavaScript engine, and executes JavaScript code outside a web browser.
Git:
Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency. Design goals of Git include speed, data integrity, and support for distributed, non-linear workflows. It is the most widely used VCS in the world today.
Cloud Platforms:
Cloud platforms such as Google Cloud, Adobe, Azure provide infrastructure and services for deploying, managing, and scaling web applications, offering solutions like virtual servers, databases, and storage.
Docker:
Docker is a set of platform as a service (PaaS) products that use OS-level virtualization to deliver software in packages called containers. It can be used to implement an entire application, including the front end, back end, and database.
