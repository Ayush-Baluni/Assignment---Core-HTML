Tags in HTML

In HTML, tags are words or letters surrounded by angle brackets(< and >). Tags are used to initialize/create elements, such as a paagraph, heading, image, hyperlink, etc.
There are various tags in HTML, which are as follows:

  1. html tag - It starts with '<html>' and ends with '</html>'. This is the container of every thing in the webpage except '<!DOCTYPE>' tag. This tag is also called the root of the webpage. This tag is fundamental to creating any HTML document, as it encompasses the entire structure and content.
  2. head tag - It starts with '<head>' and ends with '</head>'. The head element contains the meta-information about the webpage, not the content. Elements like title, link, meta. etc. can be put inside the head tag. The content within the tag does not appear directly on the web page, but it is crucial for defining how the document is processed and displayed.
  3. title tag - The title element contains the title of the page. The title is displayed in the browserâ€™s title bar (the bar at the top of the browser window), as well as in bookmarks, search engine results, and many other places. It starts with a '<title>' tag and ends with a '</title>' tag.
  4. body tag - It starts with '<body>' and ends with '</body>'. The body element appears after the head element in the page. It should contain all the content of your web page: text, images, and so on. All web pages have 1 single body element, with the exception of frameset pages, which contain frame elements instead.
  5. heading tag - It starts with '<h1>' and ends with '</h1>'. Heading tags go from h1 - h6 depending upon what size you want your heading to be. The tag format remains the same in all except the number. These tags are important for structuring the content of a web page, making it easier for users to read and for search engines to index.
  6. paragraph tag - It starts with '<p>' and ends with '&l/p>'. This element lets you create paragraphs of text. It is a block-level element so it always starts on a new line and the browsers automatically add some margin before and after the element.
  7. anchor tag - It starts with '<a' and ends with '</a>'. 'href' is the attribute of the tag that contains the URL you want to link in the anchor tag. It is one of the most important elements in a web page, as it lets you create links to other content like sites, images, references, etc.
  8. image tag - It starts with '<img' and ends with '>'. It is a self closing that that ets you insert images into your web pages. To insert an image, you first upload the image to your web server, then use the 'src' attribute to reference the image filename. The 'alt' attribute is required to display text in the place of images in case the browser can't display the image.
  9. div tag - This is a container element that you can use to add more structure to your page content. For example, you can group several lines or paragraphs, even headings in a div. These are typically used for headers and footers, sidebars, image galleries, etc. It starts with '<div>' and ends with '</div>'.
  10. table tag - This element is used to create a table in HTML and it starts with '<table>', while ending with '</table>'. It consists of several other tags as well; '<tr>' defines a row in the table, '<th>' defines the table header or the topmost row, and '<td>' defines the elements in the rest of the rows. There are many other tags in table to merge, align, space and do much more with the cells and the data in the cells.
  11. list tag - There are two lists, both of which remain almost the same in format. For unordered lists, the tag is '<ul>' and for ordered lists the tag is '<ol>'. In unordered lists there are icons as the points in list, such as bullets, squares, etc. while for ordered list there are numbers, roman numerals, alphabets, etc. To initialize an item, '<li>' tag must be used which ends with '</li>'.

Apart from these, there are text tags such as:
  1. bold tag (<b>)
  2. italic tag (<i>)
  3. highlighting tag (<mark>)
  4. strike-through tag (<del>)
  5. underline tag (<u>)
  6. superscript tag(<sup>)
  7. subscript tag (<sub>)
  8. small tag (<small>)
  9. big tag (<big>)
The format for all remains the same, with a front slash on the ending tag.
