Full Stack Web Development Tech


In Full Stack Development, there are 3 primary techs:

  1. HTML
    Primary Use Cases Key Features/Benefits
    • Making structure of webpages
    • Embedding multimedia in webpages
    • Easy to learn and use
    • Seamless Integration with other tech like CSS & JS
    • HTML5 allows semantic elements like header, footer, section etc.
  2. CSS
    Primary Use Cases Key Features/Benefits
    • To control the visual presentation, style of webpages
    • To implement responsive web design techniques
    • Makes it easier to manage code by seperating dode for visual presentation
    • It provides varied layout features using Flexbox, Grid, etc.
    • Enables faster rendering of web pages
  3. JavaScript
    Primary Use Cases Key Features/Benefits
    • To enable client-side scripting
    • Used for server-side development (back-end)
    • Can be used for both client-side and server-side development
    • Vast amount of resources, libraries and frameworks and a healthy community
    • Enables real-time updates and dynamic content loading
